# Samptak Frontend

This is the frontend application for the **Samptak** Railway Crack Detection System. It provides a modern, interactive user interface for real-time track analysis.

## Features

- **Live Analysis**: Upload track images and get instant crack detection results.
- **Real-time Metrics**: View system performance and detection accuracy.
- **Interactive UI**: Built with React, Tailwind CSS, and Framer Motion for a smooth experience.

## Getting Started

### Prerequisites

- Node.js & npm installed

### Installation

1. Navigate to the frontend directory:
   ```sh
   cd frontend
   ```

2. Install dependencies:
   ```sh
   npm install
   ```

3. Start the development server:
   ```sh
   npm run dev
   ```

   The application will be available at `http://localhost:5173`.
